Schema Markup untuk Artikel Blog

Dummy Artikel · 3 menit baca · Update template: April 2026

Artikel ini adalah contoh dummy untuk mempersiapkan implementasi schema pada konten blog. Jika nanti artikel real sudah siap, bagian ini bisa diisi schema final per halaman.

Schema Dasar Article

{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Judul Artikel",
  "author": {
    "@type": "Person",
    "name": "Nama Penulis"
  }
}

Kapan Menambahkan FAQ Schema

Jika artikel memiliki blok pertanyaan-jawaban yang benar-benar terlihat untuk user, Anda dapat menambahkan FAQ schema untuk membantu mesin pencari memahami struktur konten.
