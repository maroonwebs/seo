4. Panduan Pasang Schema di Blogspot

4.1 Langkah Instalasi

  1. Update Konten: Blogger → Posts → Edit → Ganti seluruh konten dengan artikel hybrid.
  2. Update Label: Hapus noindex → Tambah: outbound-bromo, jeep-adventure, paket-korporat.
  3. Meta Description: Paste di "Search Description" (maks 155 karakter). JANGAN ubah URL.
  4. Pasang Schema: Switch ke HTML view → Paste kode JSON-LD di paling atas → Save.
  5. Optimasi Gambar: Alt text format: "[aktivitas] [lokasi] [kota]". Kompres ke WebP <150KB.
  6. Internal Link: Minimal 2 ke halaman layanan (anchor text natural).
  7. Request Indexing: Google Search Console → URL Inspection → Request Indexing.
  8. Validasi: search.google.com/test/rich-results → Pastikan FAQPage valid.

4.2 Troubleshooting & Validasi

Error UmumPenyebabSolusi
Missing field '@type'Tag script tidak lengkapBungkus dengan <script type="application/ld+json">
Invalid value for '@context'URL context salahGunakan "https://schema.org"
Schema tidak terdeteksiKode terfilter theme / di luar bodyPaste di paling atas konten atau gunakan GTM
⚠️ FAQ tidak muncul di Google? Normal! Butuh 1-4 minggu. Pastikan: (1) Artikel terindeks, (2) Tidak ada duplikat di URL lain, (3) Schema sama persis dengan teks di halaman.
